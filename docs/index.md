# Duckify Portfolio

**Course:** 304 GenAI & Robotics | HES-SO Valais-Wallis | Spring Semester 2026

**Duckify** is a full-stack pipeline: a text prompt generates a 3D rubber duck model, which is 3D-printed and then painted by a 6-DOF UR3e robotic arm.

I contribute to the **robotic arm** sub-system — simulation, motion control, and cross-team interface coordination.
